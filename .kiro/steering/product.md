# Groove - Product Overview

Groove is a vinyl-themed music player application that connects to Spotify. It provides a nostalgic vinyl record player interface for browsing and playing music from streaming services.

## Core Features

- **Music Service Authentication**: OAuth integration with Spotify
- **Vinyl Player Interface**: Retro-styled UI with animated vinyl records
- **Playlist Management**: Browse, create, and modify playlists
- **Playback Controls**: Play, pause, seek, and skip functionality
- **Custom Vinyl Designs**: Personalized vinyl artwork for playlists

## User Flow

1. Splash screen with animated vinyl logo
2. Welcome screen with Spotify authentication option
3. Main player interface with vinyl visualization
4. Playlist gallery for browsing music collections

## Target Platforms

- Web application (desktop and mobile browsers)
- Backend API service for authentication and data management
